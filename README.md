# ADB (Android debug bridge) Commands:

for connect to android device :
1. enable android debugging
2. connect with usb cable your system
3. open terminal and after install adb-tools
4. adb tcpip FAVORITE-PORT


# Devices List

adb devices

# Stop Server 
adb kill-server

# Start Server
adb start-server

# ADB shell
** adb shell **
